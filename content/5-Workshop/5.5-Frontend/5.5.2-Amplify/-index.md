# 1. Deploy Frontend with Amplify

1. Go to **AWS Amplify** > **Create new app**.
2. Select **GitHub** > Select the Repository containing your React code.
3. **Build settings:** Amplify automatically detects `npm run build`.
4. Click **Save and Deploy**.
