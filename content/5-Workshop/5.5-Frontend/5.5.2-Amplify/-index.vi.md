# 1. Deploy Frontend với Amplify

1. Truy cập **AWS Amplify** > **Create new app**.
2. Chọn **GitHub** > Chọn Repo chứa code React của bạn.
3. **Build settings:** Amplify tự động nhận diện `npm run build`.
4. Bấm **Save and Deploy**.
